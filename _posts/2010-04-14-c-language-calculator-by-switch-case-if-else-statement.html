현대 프로그래밍 언어 어디서나 볼 수 있는 조건문의 두가지 큰 흐름..

if - else if - else와 switch - case.

이 역시 C언어에서 비롯된 것이다.

(그 이전 언어에도 그대로 있는지는 모르겠다만..)

다른 언어에서 보아왔던 형식들과 사용법이 일치한다.

오늘은 C언어의 조건문으로 만드는 간단한 계산기 실습을.


#include <stdio.h>

int main(void)
{
  int num1, num2;   //입력받을 두 숫자를 담는 int타입 변수
  char op;   //입력받을 연산자를 담을 char타입 변수
  int result;   //연산의 결과값을 담을 변수
  double resultDiv;    //나누기 결과값을 담을 double타입 변수
  int flag = 0;    //나누기 연산일 경우 result대신 resultDiv를 출력하도록 하기 위한 플래그 변수

  printf("input two numbers and operator : ");
  scanf("%d%c%d", &num1, &op, &num2);    //두 수와 연산자 입력받기. 정수만 받는다.

  switch(op)    //입력받은 연산자를 기준으로 switch하기
  {
    case '+':    //기준이 되는 연산자가 char타입이므로 작은 따옴표를 사용해야함. 콜론에 주의.
      result = num1 + num2;

    case '-':
      result = num1 - num2;

    case '*':
      result = num1 * num2;

    case '%':
      result = num1 % num2;

    case '/':
      resultDiv = (double)num1 / num2;
      //resultDiv는 double타입이므로 우변에 casting연산을 이용해 타입을 바꾸어 주어야함에 주의.
      flag++;
  }

  if(flag == 0)    //나눗셈을 제외한 나머지 연산은 정수타입 출력. 나눗셈만 걸러서 double 출력.
  {
    printf("result : %d", result);
  }
  else
  {
    printf("result : %lf", resultDiv);    //resultDiv는 double타입이므로 %d대신 %lf로 쓰는 것에 주의.
  }

  return 0;
}

계산기 만들땐 언제나 나눗셈이 걸림돌이다.

정수타입으로만 계산을 해도 실수타입으로 결과를 뱉어내므로 고민하게 되는데,

조심해서 처리하지 않으면 논리에러가 날 가능성이 크다.

C언어는 데이터타입에 있어서 설렁설렁 넘어가는 법이 없어서 더 신경이 쓰인다.

위에서는 나눗셈의 결과값을 담는 변수만 double타입으로 하고

출력할 때도 if문으로 걸러서 출력을 했는데 효과적인 방법인지 모르겠다.

모든 결과값을 double타입으로 담아도 되겠지만,

printf()로 출력할 때 소수점을 정리해주지 않으면 정수인데 괜히 뒤에

0들을 줄줄이 붙이고 등장하는 찜찜함이 있기도 하고,

부동소수점 방식은 어쩔 수 없는 약간의 부정확함 때문에

대단히 의외의 결과(안좋은)를 얻는 경우도 많다.

20으로 나와야할 결과가 19.9999999로 나오는 식으로.

아무튼, switch-case와 if-else문 자체의 구조는

AS나 Processing에서 보아온 것과 일치한다. 낯설지 않다.
